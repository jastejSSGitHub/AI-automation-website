CSS Code Reference

/* Apple iPad Air Style Glassmorphism Button */
.glass-cta {
  position: relative;
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(20px) saturate(180%);
  -webkit-backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.3);
  box-shadow: 
    0 8px 32px rgba(0, 0, 0, 0.1),
    inset 0 1px 0 rgba(255, 255, 255, 0.5);
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.glass-cta:hover {
  background: rgba(255, 255, 255, 0.25);
  transform: translateY(-2px);
  box-shadow: 
    0 12px 40px rgba(0, 0, 0, 0.15),
    inset 0 1px 0 rgba(255, 255, 255, 0.6);
}

/* Sticky Bottom Center Positioning */
.sticky-bottom-center {
  position: sticky;
  bottom: 2rem;
  left: 50%;
  transform: translateX(-50%);
  z-index: 50;
}

HTML Usage

<div class="sticky-bottom-center w-max mx-auto">
  <button class="glass-cta rounded-full px-6 py-3 flex items-center gap-3 text-white font-medium">
    <iconify-icon icon="lucide:play-circle" width="24"></iconify-icon>
    <span>Dive into the display</span>
  </button>
</div>

Usage Notes

  • • Works best on colorful or image backgrounds
  • • Adjust opacity values (0.15, 0.25) based on background darkness
  • • The blur(20px) creates the frosted glass effect
  • • saturate(180%) enhances colors behind the glass
  • • Use high z-index (50+) to ensure button stays on top
  • • Sticky positioning keeps it visible while scrolling
